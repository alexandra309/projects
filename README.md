## Projects Included

### Expense Tracker
A console-based app to record and manage daily expenses.
- Allows adding, viewing, and categorizing expenses  
- Calculates total and monthly spending  
- Uses file handling for data storage (`.txt` or `.csv`)

### Hangman
A word-guessing game built with Python.
- Random word selection from a predefined list  
- Displays progress and remaining attempts  
- Input validation and replay option

### Rock-Paper-Scissors
A simple game against the computer.
- Implements basic decision logic  
- Random choice generation using the `random` module  
- Keeps track of player vs computer results

### Mastermind
A code-breaking game where the player guesses a hidden color/number sequence.
- Provides feedback on correct guesses and positions  
- Uses loops, conditionals, and lists effectively  
- Includes restart option and score counter
